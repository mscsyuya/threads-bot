# Threads プロフィール案

ペルソナ「みお」(28歳・独身・東京・タロット3年・占いジプシー卒業組)の
プロフィール文を 3 パターン用意。日記型 + アフィ路線をやめ、
保存型コンテンツ中心に切り替えるための土台。

実運用時は 1 パターンを採用し、固定投稿・投稿傾向・口調を揃える。

---

## パターン A: 保存型コンテンツ重視

```
🌙 みお|28歳/東京/独身
🃏 タロット3年|占いジプシー卒業組
📚 占い知識を「保存版」でまとめます
📌 固定投稿に78枚の覚え方あり
☕ 毎晩21時/23時半に投稿
```

狙い:
- 「保存したくなる情報アカウント」というポジショニング
- 固定投稿への導線をプロフィール内で完結
- 投稿時間を明記してリピート視聴を作る

向いている運用:
- タロット78枚連載や、チェックリスト系投稿を主軸にする場合
- フォロワーの「役に立った=保存」を KPI にする場合

---

## パターン B: キャラ立ち型

```
🔮 みお|アラサー独身×タロット3年
✍ 占いに月◯万溶かして卒業した話
🌙 占いハック・あるある・本音
💌 DMでタロット相談歓迎(無料)
```

狙い:
- 「等身大の卒業組」という人物としての引っ掛かりで覚えてもらう
- 失敗談・本音という共感ドリブンの設計
- DM導線で個別関係を作りやすくする

向いている運用:
- 共感型・あるある投稿が中心の場合
- 後々コミュニティや note への誘導を想定する場合
- DM 対応の運用コストを許容できる場合

---

## パターン C: ノウハウ特化型

```
🃏 タロット解説 みお|28歳
📚 78枚の意味を毎日1枚解説中
🌙 占いを「賢く使う」ためのアカウント
✨ 占いジプシー卒業ノウハウ発信
```

狙い:
- 完全にノウハウ媒体としてのポジションを取る
- 「タロット78枚連載」を最大の差別化資産にする
- 「賢く使う」というスタンスで占い否定派にも届く文脈を作る

向いている運用:
- タロット78枚解説シリーズを中軸に据える運用
- 個人感情・日記要素を最小化したい場合
- 後々 LP やマガジンに発展させる前提の場合

---

## 採用基準のメモ

- 保存数・固定投稿のクリック率を上げたい → A
- フォロワー数・エンゲージメント率を上げたい → B
- 1 つの強いシリーズを軸に育てたい → C

迷ったら A から始めて、3 週間反応を見て切り替えるのが安全。
